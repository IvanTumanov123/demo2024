# №1.2
Сначала я устанавливаю пакеты FRR
```
apt-get install frr
```
Проверил его состояние
```
systemctl status frr
```
Далее я ввожу
```
vtysh
```
И ввожу данные команды
```
conf t
router ospf
 ospf router-id 192.168.1.1
 network 192.168.1.160/30 area 0
 network 192.168.2.164/30 area 0
```
